This screen is a live UI preview. In a real run the agent generates SPL from the chosen control, validates it through the policy gate, and executes it against Splunk BOTS v3 via the MCP server. Four independently-trained models — Claude, GPT, Gemini and Splunk's Foundation-Sec-8B — then deliberate over the returned events and reach a mechanical consensus (lowest verdict wins, no LLM tiebreaker, fully reproducible). The verdict is SHA-256 Merkle-chained into a tamper-evident trail and written into the xlsx audit tracker. The ▶ Watch the real run link is a recording of that exact CLI pipeline.
